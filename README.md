# Division Using Promises in JavaScript

## Project Description

This project demonstrates how to perform division using JavaScript Promises. The program takes two numbers as input and returns the result of the division. It also handles errors such as division by zero using Promise rejection.

## Features

* Uses JavaScript Promises.
* Uses Arrow Functions.
* Handles division by zero errors.
* Displays appropriate success and error messages.
* Includes 5 test cases for demonstration.
* Simple and beginner-friendly code structure.

## Project Files
1. **index.html** - Structure of the webpage.
2. **style.css** - Styling for the webpage.
3. **script.js** - Promise logic and test cases.

## How It Works

1. The user clicks the **Run Test Cases** button.
2. The JavaScript function performs division using a Promise.
3. If the divisor is zero, the Promise is rejected with an error message.
4. Otherwise, the Promise is resolved with the division result.
5. The output is displayed on the webpage.
